# PWA Icons

This directory should contain PNG icons for your PWA. You need the following sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these from the provided icon.svg using tools like:
- Online SVG to PNG converters
- ImageMagick: `convert icon.svg -resize 512x512 icon-512x512.png`
- Figma or other design tools

For now, the PWA will work with placeholder icons, but for production you should add proper PNG icons.
